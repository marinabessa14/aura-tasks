 Aura-Tasks — Gerenciador de Tarefas Inteligente

Aplicação full-stack que usa Inteligência Artificial para categorizar tarefas e estimar o tempo de execução automaticamente.

-----------------------------------------------------------------------------------------------------------------------------------------

Sobre o projeto

O Aura-Tasks é um gerenciador de tarefas que vai além do To-Do List comum. Ao adicionar uma tarefa, a IA analisa o título e automaticamente:

Define a categoria (Trabalho, Estudos, Saúde, Casa, etc.)
Estima o tempo de execução (15 minutos, 1 hora, etc.)

Tudo salvo em banco de dados real na nuvem.

------------------------------------------------------------------------------------------------------------------------------------------------- 
Funcionalidades:

Adicionar tarefas com categorização automática por IA;

Estimativa de tempo gerada automaticamente;

Marcar tarefas como concluídas;

Deletar tarefas;

Filtrar tarefas por categoria;

Barra de progresso mostrando % de conclusão;

Dados persistidos em banco de dados na nuvem;

Layout responsivo para mobile.

-----------------------------------------------------------------------------------------------------------------------------------

Tecnologias utilizadas:

Next.js 14: Framework React com App Router e API Routes;

React: Interface do usuário com hooks (useState, useEffect);

Supabase: Banco de dados PostgreSQL na nuvem;

Groq + LLaMA 3.1: IA para categorização e estimativa de tempo;

CSS Modules: Estilização com escopo isolado por componente.
